> **The Real Problem Campus Students Face:**
>
> тЭМ Off-campus jobs scattered across 10+ platforms тАФ apply everywhere separately  
> тЭМ On-campus drives happen over Gmail тАФ placement mails get buried, go unseen, deadlines missed  
> тЭМ No single place to track both on-campus and off-campus opportunities together  

> **HireSync fixes all three тАФ one platform, everything in one place.**

```

ЁЯУД  Resume upload      тЖТ  Auto skill extraction from PDF
ЁЯПл  Campus drives      тЖТ  No more lost placement Gmail threads

ЁЯФФ  Real-time notifs   тЖТ  Socket.IO тАФ zero missed updates
```
