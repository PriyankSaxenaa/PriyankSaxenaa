## 🚀 Featured Project: HireSync

### The Problem
Campus hiring is completely broken —

📌 **Off-campus chaos:** Students bounce between 10+ job platforms  
📌 **Email hell:** On-campus drives happen via Gmail → placement mails get buried and go unseen  
📌 **No visibility:** Zero single place to track both on-campus drives and off-campus opportunities  

**Result?** Students miss deadlines, miss opportunities, waste hours switching tabs.

### The Solution
**HireSync** — One unified platform where everything lives in one place.

```
👥 Multi-role system        → Candidate, Recruiter, TPO, Admin
📄 Resume upload            → Auto skill extraction from PDF
🏫 Campus drives            → No more lost placement emails
📋 Bulk imports             → Excel/CSV student onboarding
🔔 Real-time notifications → Socket.IO push (never miss an update)
💡 Smart recommendations    → Jobs matched to your skills
📊 Full analytics           → Drive stats, application tracking
🔐 RBAC security           → Locked down by role
```

### Tech Stack
```
Backend    → Node.js, Express.js, MongoDB, Redis
Real-time  → Socket.IO, Nodemailer
Files      → Cloudinary, Multer, PDF Parsing
Deployment → Render
```
